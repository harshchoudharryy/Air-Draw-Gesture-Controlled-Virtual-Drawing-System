About The Project

Project Overview

Air Canvas is a Computer Vision application that enables users to draw, erase, and interact with a virtual canvas using only hand gestures captured through a webcam.

The project uses OpenCV and MediaPipe to detect hand landmarks in real-time and convert gestures into drawing actions.
